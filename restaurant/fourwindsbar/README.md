##### Create Project
`npm create vite@latest fourwindsbar` <br>

##### Select folder
`cd fourwindsbar` <br>

##### Install the dependecies
`npm install` <br>
`npm install -D tailwindcss` <br>
`npx tailwindcss init` <br>

##### To run app
`npm run dev` <br>
